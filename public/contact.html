자주 묻는 질문

이런 질문이 많이 들어와요

탄생별 결과가 이상해요. 오류인가요?

탄생별은 생년월일 기준 나이(년)와 같은 거리(광년)에 있는 별 중 가장 밝은 별을 찾는 방식입니다. 동일한 거리에 여러 별이 있을 수 있어 결과가 다르게 나오는 경우가 있습니다. GitHub Issues에 날짜와 함께 제보해 주세요.

사용한 별 데이터는 어디서 왔나요?

Yale Bright Star Catalogue(예일 밝은별 목록)와 Hipparcos 성표 데이터를 기반으로 합니다. 거리 데이터는 시차(parallax) 측정값에서 추출했습니다.

아티클 주제 제안을 해도 되나요?

물론입니다. 별, 우주, 천문학에 관한 궁금한 주제가 있으시면 GitHub Issues에 남겨주세요. 다음 아티클 주제 선정에 반영하겠습니다.

광고나 협업 문의는 어떻게 하나요?

GitHub 프로필 페이지를 통해 연락 주시면 확인하겠습니다.
